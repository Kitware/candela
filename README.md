# Candela
Visualization components for the web

# [Documentation](src/candela#readme)

# Installation from source

## Prerequisites

* npm
* cairo (`brew install cairo` on Mac OSX).

## Build

```bash
git clone https://github.com/Kitware/candela.git
cd candela
npm install
npm run build
```
